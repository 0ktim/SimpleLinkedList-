# SimpleLinkedList

🚀 **Персонализирана имплементация на едносвързан списък в C#**

Този проект съдържа **опростена имплементация на едносвързан списък** (Linked List) в езика C#. Реализацията включва основни операции като добавяне, премахване, достъп до елементи и проверка дали даден елемент съществува в списъка.

## Основни функции
- ✅ **Добавяне на елементи** към списъка
- ✅ **Премахване на елемент** по индекс
- ✅ **Достъп до елемент** по индекс
- ✅ **Проверка дали елемент съществува** в списъка
- ✅ **Извеждане на списъка** в конзолата

## Употреба
### Примерен код:
```csharp
LinkedList list = new LinkedList();

list.Add(10);
list.Add(20);
list.Add(30);
list.Add(40);

list.Print(); // Изход: 10 -> 20 -> 30 -> 40 -> null

Console.WriteLine(list.Get(2)); // Изход: 30

list.RemoveAt(1); // Премахва 20
list.Print(); // Изход: 10 -> 30 -> 40 -> null

Console.WriteLine(list.Contains(30)); // Изход: true
Console.WriteLine(list.Contains(70)); // Изход: false
